How to set it up?
===================


  - Install python on your machine
  - install Flask with pip
  ```sh
  $ pip install flask
  ```
  - Navigate to the directory with project
  ```sh
  $ export FLASK_DEBUG=1
  $ export FLASK_APP=1
  $ flask run
  ```
  - That's it!
